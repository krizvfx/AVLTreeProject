# AVL Tree Project (C#)

## Кратко описание на задачата
Проектът реализира структура от данни **AVL дърво**, което поддържа елементите сортирани и автоматично се балансира след операции като добавяне и изтриване. Програмата демонстрира основните операции върху дървото и извежда резултатите в конзолата.

## Реализиран тип дърво
- **AVL дърво (самобалансиращо се двоично дърво за търсене)**

## Използван програмен език и технологии
- **C#**
- **.NET (Console Application)**
- Visual Studio 2022

## Как се стартира програмата
1. Отворете проекта във Visual Studio.
2. Стартирайте приложението:
   - Visual Studio: `Start` / `Ctrl + F5`

## Пример за работа (вход/изход)

### Вход (примерни добавени стойности)
Програмата добавя следните елементи в дървото:
```
14, 7, 31, 11, 20, 10
```

### Изход (пример)
```
InOrder обхождане:
7 10 11 14 20 31

Търсене на 7: True
Търсене на 67: False

След изтриване на 10:
7 11 14 20 31
```
